# getDeviceState

#### 接口功能
> 获取当前设备连接状态

#### 函数原型

```objective-c
-(NSInteger) getDeviceState;
```

#### 参数
> 空

#### 返回值
| 参数类型 | 参数说明 |
| :--------| :------ |
| NSInteger | 当前设备连接状态(0: 设备未连接, 1: 设备已连接) |
